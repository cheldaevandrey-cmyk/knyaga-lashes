# Knyaga Lashes

Лендинг для мастера по наращиванию ресниц. Мобильный дизайн в редакционном стиле — розовая палитра, Playfair Display, газетная вёрстка.

## Что внутри

- **Услуги и прайс** — классика, 2D, 3D, 4D, ламинирование ресниц и бровей
- **Портфолио** — фото работ мастера
- **Онлайн-запись** — форма с выбором услуги
- **SEO** — мета-теги, Open Graph, Twitter Card

## Стек

- HTML + CSS + Vanilla JS — без фреймворков
- Google Fonts: Playfair Display, Special Elite, DM Sans
- Адаптив под мобильные (max-width: 480px)
- Деплой: Vercel

## Деплой

```bash
# Vercel CLI
vercel --prod
```

Или просто открыть `index.html` в браузере.
